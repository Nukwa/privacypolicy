Politique de Confidentialité

Dernière mise à jour : 15 janvier 2024

1. Introduction

Bienvenue sur Meub Loc Appart (« nous », « notre », « nos »). Nous accordons une grande importance à la protection de votre vie privée et de vos données personnelles. Cette politique de confidentialité explique quelles données nous collectons, comment nous les utilisons et les protégeons lorsque vous utilisez notre application de location d'appartements meublés et non meublés.

En utilisant Meub Loc Appart, vous acceptez les pratiques décrites dans cette politique de confidentialité.

2. Données que nous collectons

2.1. Informations que vous nous fournissez :

  • Données de profil : Nom, prénom, adresse e-mail, numéro de téléphone, photo de profil
  • Données de propriétaire : Documents d'identité pour vérification, informations bancaires pour paiements
  • Données des biens : Photos, descriptions, adresses, caractéristiques des appartements
  • Données de transaction : Historique des réservations, paiements, communications

2.2. Informations collectées automatiquement :

  • Données d'utilisation : Logs de connexion, pages visitées, fonctionnalités utilisées
  • Données techniques : Type d'appareil, système d'exploitation, version de l'application
  • Données de localisation : Uniquement avec votre consentement pour la recherche de biens à proximité

3. Utilisation de vos données

Nous utilisons vos données pour :

  • Faciliter la mise en relation entre propriétaires et locataires
  • Vérifier l'identité des propriétaires bailleurs
  • Traiter les paiements et transactions sécurisées
  • Personnaliser votre expérience utilisateur
  • Vous envoyer des notifications importantes
  • Améliorer notre application et nos services
  • Prévenir les fraudes et assurer la sécurité

4. Partage de vos données

Nous ne vendons pas vos données personnelles. Nous pouvons partager vos informations avec :

  • Utilisateurs de l'application : Propriétaires voient les informations des locataires et vice-versa dans le cadre des réservations
  • Prestataires de services : Processeurs de paiement, services d'hébergement, support technique
  • Obligations légales : Lorsque requis par la loi ou pour protéger nos droits

5. Conservation des données

Nous conservons vos données :

  • Données de compte : Tant que votre compte est actif + 3 ans après désactivation
  • Données de transaction : 10 ans pour conformité légale
  • Données de localisation : Immédiatement anonymisées après utilisation

6. Vos droits

Conformément au RGPD, vous avez le droit :

  • D'accéder à vos données personnelles
  • De rectifier les données inexactes
  • De supprimer vos données (« droit à l'oubli »)
  • De limiter le traitement de vos données
  • À la portabilité de vos données
  • De vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous à : privacy@meublocappart.com

7. Sécurité des données

Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles appropriées :

  • Chiffrement des données sensibles
  • Authentification sécurisée
  • Contrôles d'accès réguliers
  • Sauvegardes régulières
  • Formation du personnel à la protection des données

8. Cookies et technologies similaires

Notre application utilise :

  • Cookies essentiels : Pour le fonctionnement de l'application
  • Cookies analytiques : Pour analyser l'usage et améliorer nos services
  • Cookies de préférences : Pour mémoriser vos choix

9. Paiements sécurisés

Les transactions financières sont traitées par des prestataires de paiement certifiés PCI-DSS. Nous ne stockons pas les informations de carte bancaire.

10. Modification de cette politique

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications significatives seront notifiées par :

  • Notification dans l'application
  • E-mail aux utilisateurs
  • Date de "dernière mise à jour" modifiée
